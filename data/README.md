# Waste Classfication Dataset

This dataset contains images of various types of waste, categorized into different classes such as cardboard, glass, metal, paper, plastic, and trash. It is intended for use in machine learning projects focused on waste classification and recycling.

The dataset can be downloaded from: [https://www.kaggle.com/datasets/kaanerkez/waste-classfication-dataset/data](https://www.kaggle.com/datasets/kaanerkez/waste-classfication-dataset/data)
