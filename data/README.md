# Electronic Components Dataset

This dataset contains images of various electronic components such as resistors, capacitors, inductors, diodes, and transistors. It is intended for use in machine learning and computer vision projects related to component recognition and classification.

The dataset can be downloaded from: [https://www.kaggle.com/datasets/aryaminus/electronic-components/data](https://www.kaggle.com/datasets/aryaminus/electronic-components/data)
